\# Supermarket Sales Analysis using R



\## Project Overview



This project analyzes supermarket sales data using R. The analysis includes data cleaning, exploratory data analysis (EDA), data visualization, and regression modeling to understand sales patterns and predict tax values.



\## Features



\* Data Cleaning and Preprocessing

\* Missing Value Analysis

\* Duplicate Detection

\* Outlier Handling

\* Customer and Product Analysis

\* Data Visualization using ggplot2

\* Regression Model Building using caret



\## Tools \& Technologies



\* R

\* RStudio

\* dplyr

\* readr

\* ggplot2

\* caret



\## Dataset



Supermarket Sales Dataset
