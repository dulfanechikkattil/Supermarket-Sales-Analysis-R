# 🛒 Supermarket Sales Analysis using R

## 📌 Project Overview

This project analyzes supermarket sales data using R to uncover customer purchasing behavior, product demand trends, sales performance, and tax patterns. The project includes data cleaning, exploratory data analysis (EDA), data visualization, and regression modeling.

## 🚀 Key Features

* Data Cleaning & Preprocessing
* Missing Value Analysis
* Duplicate Detection
* Outlier Handling
* Customer Behavior Analysis
* Product Demand Analysis
* Data Visualization using ggplot2
* Regression Modeling using caret

## 🛠️ Tools & Technologies

* R
* RStudio
* dplyr
* readr
* ggplot2
* caret

## 📊 Project Insights

* Identified customer spending patterns.
* Analyzed product demand across categories.
* Explored tax contribution trends.
* Developed a regression model for tax prediction.
